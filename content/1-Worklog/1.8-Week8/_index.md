### 3. Results Achieved

During Week 8, I moved from general cloud learning into project-oriented AWS planning. I studied how to draw AWS architecture diagrams and mapped each project function to a suitable cloud service.

For the final project, my main contribution was to prepare the AWS service architecture. The planned services included S3 and CloudFront for frontend hosting, Cognito for authentication, API Gateway and Lambda for backend APIs, EventBridge for scheduled ingestion, S3 for raw data storage, DynamoDB for storing processed results, Bedrock for AI analysis, and CloudWatch for monitoring.

The study session on June 11 helped me understand the project requirements more clearly. After this session, I had a better view of which parts belonged to frontend and backend members, and which AWS services I needed to configure for the cloud infrastructure side.
