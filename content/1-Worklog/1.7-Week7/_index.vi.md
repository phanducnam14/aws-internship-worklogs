### 3. Results Achieved

This week helped me connect separate AWS services into a bigger system view. Instead of only learning each service individually, I started thinking about how services can work together in a real project.

AWS WAF helped me understand that cloud security is not only about IAM or network rules. Web applications also need protection from application-layer attacks such as SQL Injection and XSS.

For the final project, the team started discussing an AI-based system using AWS services. My direction in the group was to focus more on AWS infrastructure setup and cloud service configuration for the project.
