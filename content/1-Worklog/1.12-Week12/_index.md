# Overall Internship Summary

Throughout the First Cloud AI Journey program, I gained hands-on experience with many important AWS services and cloud concepts. I started from basic cloud knowledge such as AWS Global Infrastructure, IAM, EC2, S3, and Billing, then gradually moved to networking, security, automation, serverless architecture, and AI service integration.

For the final group project, my main contribution focused on AWS service setup and cloud infrastructure integration. I supported the team by designing the AWS architecture, preparing service mapping, configuring S3 and CloudFront for frontend hosting, reviewing API Gateway and Lambda integration, planning EventBridge scheduled tasks, preparing DynamoDB and Bedrock connection direction, and using CloudWatch for troubleshooting.

This internship helped me understand that building a cloud system is not only about deploying services, but also about security, cost control, monitoring, architecture design, and teamwork. It also improved my mindset toward DevOps and Cloud Engineering, especially in the areas of AWS infrastructure setup, service integration, and system troubleshooting.
