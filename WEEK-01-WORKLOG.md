### 3. Results Achieved

During the first week, I gained a clearer understanding of AWS Cloud fundamentals and how to prepare a safe practice environment. I created an AWS account, enabled MFA, configured budget alerts, and explored several basic AWS services.

The most important lesson was that cloud learning must always go together with security and cost control. Before deploying any resource, it is necessary to understand account permission, service pricing, and cleanup steps.

I also learned that IAM is one of the most important foundations in AWS. Instead of using root account access or permanent credentials, each user and service should be granted only the permissions required for its task.
